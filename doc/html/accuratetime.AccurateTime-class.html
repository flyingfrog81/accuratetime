Class AccurateTime

source code

Class for storing a POSIX time with arbitrary floating point precision. It can return datetime objects as well as posix time and the values are kept synchronized between each other. In order to keep consisency between datetime object and posix time information, changes must be made using datetime and accurate_posix public attributes.
